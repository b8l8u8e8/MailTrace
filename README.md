# MailTrace 📧

一个轻量的 PHP 邮件追踪系统，用来生成追踪代码、记录访问日志，并支持邮件提醒。

## 它是干什么的 ✨

- 生成邮件追踪代码（多种嵌入方式）
- 记录访问日志（时间、IP、地区、UA）
- 命中后可发送提醒邮件
- 支持后台管理用户、邀请码、注册开关等

## 主要功能 🧩

- 登录/注册/邮箱激活
- 用户面板：创建和管理追踪码
- 日志查看与清理
- 管理后台：SMTP 配置、邀请码管理、用户管理

## 宝塔面板部署示例 🚀

1. 宝塔里新建网站（PHP 项目），域名或 IP 都可以。  
2. 把项目文件上传到站点根目录（例如 `www/wwwroot/mailtrace`）。  
3. 确认 PHP 版本可用（建议 PHP 7.4+，推荐 8.x）。  
4. 访问站点一次，系统会自动创建 `data.db`。  
5. 到后台配置 SMTP（用于激活邮件和提醒邮件）。  
6. 伪静态/配置里加入下面规则，禁止数据库文件被下载：

```nginx
location ~ .*\.(db)$ {
    deny all;
}
```

## 默认管理员账号 🔐

首次初始化数据库时默认账号：

- 用户名：`admin`
- 密码：`admin123`

上线后请第一时间修改密码。

## 其他环境说明 🌍

除了宝塔，任何能运行 PHP 的环境都可以部署，例如：

- Nginx + PHP-FPM
- Apache + PHP
- 宝塔以外的 Linux 面板
- 本地 `php -S` 临时运行
